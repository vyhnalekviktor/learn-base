Receive Funds

Learn how to receive USDC on Base network

Your wallet address is your unique identifier on the blockchain. Here's how to find it:

  1. Open your Base Wallet or Coinbase Wallet app
  2. Tap on your profile or username at the top
  3. Look for your wallet address - a 42-character code starting with "0x"
  4. Tap the address to copy it to your clipboard

It really is that simple!

Note: Your address is the same on both mainnet and testnet. The difference is which network the sender uses when sending funds.

Send your wallet address to the person or platform that will send you USDC. You can share it by:

  • Copying and pasting the address
  • Showing a QR code (available in most wallets)
  • Sharing your Base username (if using Base Wallet)

Important: Make sure the sender knows to use the Base network, not Ethereum or another chain. USDC sent on the wrong network won't appear in your Base wallet!

Once the sender initiates the transaction, you'll typically see the USDC in your wallet within 1-5 seconds. Base processes transactions much faster than Ethereum mainnet.

You can verify the transaction by:

  • Checking your wallet's activity history
  • Viewing it on BaseScan explorer (mainnet) or Sepolia BaseScan (testnet)

If you don't see your USDC after a few minutes, try these troubleshooting steps:

In Your Base Wallet:

  • Check if you have any networks filtered out - make sure Base network is visible
  • If using testnet, go to Settings and enable "Show testnet balances"
  • Try refreshing your wallet or closing and reopening the app

Verify with Sender:

  • Confirm they used the correct wallet address (ask them to double-check)
  • Make sure they sent on Base network, not Ethereum or another chain
  • Ask for the transaction hash to verify on BaseScan

Still having issues? Check the transaction on BaseScan using the transaction hash. It will show if the transaction was successful and which address received the funds.
