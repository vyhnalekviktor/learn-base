BaseCamp Crypto Glossary

Complete guide for total beginners. Every important term explained simply, step by step.

Blockchain
Think of blockchain as a public accounting book managed by thousands of computers worldwide. Every transaction (like sending money) gets permanently written here and nobody can change it. On Base, transactions process fast and cheap, then get batched and sent to Ethereum for security.
Testnet
Testing version of blockchain where you use fake money with zero risk. Base Sepolia works exactly like mainnet but nothing costs real money. Perfect practice ground for all your labs.
Mainnet
Live network with real money. Every transaction here is permanent and can't be undone. Be very careful what you sign!
Transaction
Any sending of money, NFT, or running code on blockchain. Like a bank transfer but public and irreversible. Everyone can see it on block explorer.
Gas Fee
Tiny ETH payment to process transactions - like gas for network computers. Base makes them 95-99% cheaper than Ethereum. You pay for sends, mints, and swaps.
Wallet
App or device holding your private key and address. Doesn't store money directly - just lets you send/receive. Like digital bank account but YOU have full control.
Wallet Address
Long string starting with 0x - your unique account number. Send money here. Share freely but always double-check before sending.
Private Key
Your most important secret - password proving money belongs to you. NEVER share it! Wallet uses it to sign transactions without showing it.
Signature
Code created from your private key confirming transactions. Like your handwritten signature but cryptographically secure - nobody else can fake it.
Seed Phrase / Recovery Phrase
12-24 words to recover your wallet. Backup of your private key. Write on paper, store safely, NEVER enter on websites - that's always a scam!
USDC
Stablecoin always worth $1. Perfect for practice - free from faucets on testnet. You use it in labs for sending and swaps.
ETH
Base currency of Ethereum and Base. Pays gas fees. Free from faucets on testnet, buy real ETH for mainnet.
ERC-20
Standard for interchangeable tokens like USDC. All ERC20 tokens work the same - send like regular money.
ERC-721
Standard for unique items. Each NFT is different, like collectible trading cards. Your mint.html creates ERC721 NFTs with SVG images.
NFT
Non-Fungible Token = unique digital asset. Proof of ownership for images, videos, or collectibles. Created by calling smart contracts.
Token ID
Number for each NFT (1, 2, 3...). Shows creation order. After minting, fetch it from contract's counter().
Base
Fast, cheap network built by Coinbase on Ethereum (Layer 2). Processes transactions in 1-2 seconds, batches them, sends to Ethereum for security. Core to all your labs.
Base Sepolia
Base testnet (chainId 84532). Used in all practice files for safe learning. Try Base Sepolia explorer
Faucet
Website giving free test tokens after captcha/address. Circle for USDC, Alchemy for ETH. Search "Base Sepolia faucet".
DEX (Decentralized Exchange)
Decentralized exchange for token swaps without banks. lab5.js simulates fake SuperSwap vs real Coinbase rates.
Block Explorer
Website to track transactions: sepolia.basescan.org. Paste hash or address to see details. Check your mints/sends here.
Transaction Hash
Unique 66-character ID (0x...) for every transaction. Paste into explorer to track status.
Sequencer
Base's special computer that instantly processes transactions off-chain, then batches to Ethereum. Makes Base super fast.
WalletConnect
Secure wallet-web connection via QR code. Only safe method in lab4.js - never type keys manually!
Smart Contract
Automatic program on blockchain like a vending machine: input condition, get result. Your NFT contract 0x726107... mints NFTs when called.
Mint
Creating new NFT by calling contract function (mintTo). Uploads to blockchain with your address as owner.
Metadata
NFT description: name, image, text in JSON format. Your contract generates SVG directly on-chain as base64.
Token URI
Link to NFT metadata. Your contract returns base64 data directly, no external servers needed.
On-chain
Data stored directly on blockchain - forever and decentralized. Your NFTs store full SVG/metadata on-chain, no IPFS needed.
